# Model Construction and Statistical Inference

Often times machine learning is treating as a grab-bag of seemingly
unrelated techniques: supervised learning, unsupervised learning,
dimensionality reduction, etc. The differences between these techniques
are trivial from a probabilistic perspective. Our approach for all of
these problems is very simple. We begin with some dataset $\mathcal{D}$.

This dataset may be comprised of input-output pairs. For example, we may
have $\mathcal{D} = \{(x_i, y_i)\}_{i = 1}^N$. This called supervised
learning.

This dataset can also just be comprised of single data points, i.e.
$\mathcal{D} = \{x_i\}_{i = 1}^N$. The problem of finding the
probability distribution which generated $\mathcal{D}$ is called density estimation. This is an example of unsupervised learning but is only trivially different from supervised learning when this probabilistic approach is taken.

In each scenario, we begin by assuming a *causal story* for how our data was generated by specifying an **observation model**. In Bayes Rule, we have:

$$
p(\theta \mid \mathcal{D}) = \frac{p(\mathcal{D} \mid \theta) p(\theta)}{p(\mathcal{D})}
$$

Here $\theta$ (occasionally written as $a, w$) are the parameters of our observation model. For example, if we assume a linear observation model,
we have:

$$
y_i = a x_i + \epsilon_x
$$

where $\epsilon_x$ is some irreducible noise term.

The process of inferring parameters like $\theta$ is called . This can
be done by finding the joint distribution, $p(D, \theta)$. Once we have
the joint distribution, through the sum and product rules, we can
recover the marginal and conditional distribution.

$$
p(D) = \int p(D \mid \theta) p(\theta) d\theta
$$
$$
p(D \mid \theta) = \frac{p(D, \theta)}{p(\theta)}
$$

Bayesian inference allows us to faithfully represent our uncertainty
over the parameters $\theta$. Even if we find a point estimate
$\hat{\theta}$ which maximizes the likelihood of the data given
$\theta$, i.e. $p(\mathcal{D} \mid \theta)$, given a limited amount of
data, we still have uncertainty over the true value of $\theta$.

## MLE

**MLE** or **Maximum Likelihood Estimation** is a procedure for
inferring the parameters of some model.

Let $\mathcal{D} = \{(x_i, y_i)\}_{i = 1}^n$. We begin by specifying an observation model that relates $y_i$ and $x_i$ through some function
$f(x, a) = ax$.

::: centeredboxgreen
**Model Classes**

The observation model is generally a - specific models within this model
class are specified by some parameter value $\theta$. We always assume
that the ground truth is an element of this model class, i.e. $y_i$ and
$x_i$ truly do have some linear relationship. For simplicity's sake, we
're using a linear model but in reality, we often want much more
flexible models. We will examine this in further detail in the model
selection section.
:::

So we have: $$\begin{equation}
\label{eq:obs_1}
    y = ax + \epsilon_x
\end{equation}$$ where $\epsilon_x \sim \mathcal{N}(0, \sigma^2)$ is our
Gaussian noise term. We want to know how likely it is that our model
could have generated this dataset. Specifically, the is defined as:
$$\begin{equation}
\theta_{\text{ML}} = \mathop{\mathrm{arg\,max}}_{\theta} p(\mathcal{D} \mid \theta)
\end{equation}$$

We begin by computing the likelihood that the data was generated by our
model, i.e. $p(\mathcal{D} \mid \theta)$. Since the only random term in
$(\ref{eq:obs_1})$ is $\epsilon_x$, the random variable $y$'s
distribution is just a displaced version of $\epsilon_x$'s distribution.
We have:

$$\begin{equation}
    y \mid a, x, \sigma^2 \sim \mathcal{N}(y ; ax, \sigma^2)
\end{equation}$$ $$\begin{equation}
    p(y \mid x, a, \sigma^2) = \frac{1}{\sqrt{2 \pi \sigma^2}} \exp \left \{ -\frac{1}{2 \sigma^2} (y - ax)^2 \right \}
\end{equation}$$

However, this is the likelihood of a single sample $(x_i, y_i)$. We want
the likelihood of the entire dataset. Let
$\mathbf{y} = (y_1, \dots, y_n)^T, \mathbf{X} = (x_1, \dots, x_N)^T$. We
can assume that $\mathbf{y}$ is **conditionally independent** on the
parameters $a, \sigma^2$.

:::: centeredboxblue
::: example
**Example: 1**. **Conditional Independence**

Suppose we have a normal distribution $\mathcal{N}(\mu, 1)$. The first
sample I draw from this distribution has value $1000000$. Does this
influence your beliefs about the second sample I will draw? It should.
Clearly, sample 1 and 2 are not completely independent.

If the distribution is $\mathcal{N}(1000001, 1)$ and the first sample I
draw is $1000000$, does that influence your beliefs about the second
sample? It shouldn't really. These samples are **conditionally
independent** once the parameters of the distribution are known.
:::
::::

We want to find the likelihood of the entire dataset,
$p(y_1, \dots y_n \mid x_1, \dots, x_n, \sigma^2, a) = p(\mathbf{y} \mid \mathbf{x}, a, \sigma^2)$.
We have: $$\begin{align}
p(\mathbf{y} \mid a, \mathbf{X}, \sigma^2) &= \prod_{i = 1}^n p(y_i \mid a, x_i, \sigma^2) \\
&= \prod_{i = 1}^n \frac{1}{\sqrt{2 \pi \sigma^2}} \exp \left \{
- \frac{1}{2 \sigma^2} (y_i - ax_i)^2
\right \} \\
&= \frac{1}{(2 \pi \sigma^2)^{\frac{n}{2}}} \exp \left \{
- \frac{1}{2 \sigma^2} \sum_{i = 1}^n (y_i^2 - 2 a x_i y_i + a^2 x_i^2)
\right \} \\
&= \frac{1}{(2 \pi \sigma^2)^{\frac{n}{2}}} \exp \left \{
- \frac{1}{2 \sigma^2} (\mathbf{y}^T \mathbf{y} - 2 a \mathbf{x}^T \mathbf{y} + a^2 \mathbf{x}^T \mathbf{x})
\right \}
\end{align}$$ A common trick is to take the log-likelihood (this
could've been done earlier to convert the product to a sum but this also
works). $$\begin{align}
    \log p(\mathbf{y} \mid a, \mathbf{X}, \sigma^2) = - \frac{n}{2} \log (2 \pi) - \frac{n}{2} \log (\sigma^2) - \frac{1}{2\sigma^2} (\mathbf{y}^T \mathbf{y} - 2 a \mathbf{x}^T \mathbf{y} + a^2 \mathbf{x}^T \mathbf{x})
\end{align}$$ Now, by taking derivatives we can find maximum likelihood
estimators. $$\begin{align}
    0 = \frac{\partial \log p(\mathbf{y} \mid a, \mathbf{X}, \sigma^2)}{\partial a} &= \frac{d}{da} 2 a \mathbf{x}^T \mathbf{y} - \frac{d}{da} a^2 \mathbf{x}^T \mathbf{x} \\
    2 a \mathbf{x}^T \mathbf{x} &= 2 \mathbf{x}^T \mathbf{y}  \\
   \Aboxed{a_{\text{ML}} &= \frac{\mathbf{x}^T \mathbf{y}}{\mathbf{x}^T \mathbf{x}}}
\end{align}$$ $$\begin{align}
    0 = \frac{\partial \log p(\mathbf{y} \mid a, \mathbf{X}, \sigma^2)}{\partial \sigma^2} &= - \frac{n}{2 \sigma^2} + \frac{1}{2 \sigma^4} (\mathbf{y} - a \mathbf{x})^2 \\
    \Aboxed{\sigma^2_{\text{ML}} &= \frac{1}{n} (\mathbf{y} - a_{\text{ML}} \mathbf{x})^2}
\end{align}$$

:::: centeredboxgreen
::: intuition
**Intuition Check: 1**. **Likelihood**

The **likelihood**, $p(D \mid a)$ is the probability of observing some
data points $D$ given a parameter setting $a$. The **maximum likelihood
estimators** are the values which maximize this, i.e.
$\hat{a} = \mathop{\mathrm{arg\,max}}_a p(D \mid a)$. Intuitively, if we
set our parameters to a specific $a_0$ and our likelihood decreases, we
know $a_0$ is unlikely to be the correct parameter setting (or
equivalent to $a$ in the observation model). Maximizing likelihood is a
*frequentist* method but isn't always the correct solution.
:::
::::

The MLE for the coefficient in the linear regression problem is the
standard result -
$\hat{a} = \frac{\text{Cov}(X, y)}{\mathop{\mathrm{Var}}(X)} = (X^T X)^{-1} X^T y$
. The MLE for the variance of the noise term is the mean-squared-error -
MSE can be written as the sum of bias + variance and since the bias of
the error term is $0$ (as it has $0$ mean), the MSE is directly
equivalent to the variance $\sigma^2$.

:::: centeredboxgreen
::: intuition
**Intuition Check: 2**. **Likelihood**

The likelihood can be written as
$p(\mathbf{y} \mid \mathbf{x}) = \prod_{i = 1}^n p(y_i \mid x_i)$
:::
::::

## MAP

Maximizing $p(D \mid a)$ as the likelihood is maximizing the probability
of observing $D$ given $a$. But we care about setting our parameters
correctly, not just observing $D$. **Maximum a posteriori estimation**
is about maximizing the posterior probability, or the probability of the
parameters given the data. Specifically, the MAP is:

$$\hat{a} = \mathop{\mathrm{arg\,max}}_a p(a | D) \propto \mathop{\mathrm{arg\,max}}_a p(D | a) p(a)$$
$$= \mathop{\mathrm{arg\,max}}_a \log p(a | D) = \mathop{\mathrm{arg\,max}}_a \log p(D | a) + \mathop{\mathrm{arg\,max}}_a \log p(a)$$

If we place a prior distribution over $a \sim \mathcal{N}(0, \gamma^2)$,
then we have:
$$p(a \mid \gamma^2, \mathbf{y}, X, \sigma^2) = p(\mathbf{y} \mid X, \sigma^2, a) \ p(a \mid \gamma^2)$$
$$\log p(a \mid D, \gamma^2) = \log p(\mathbf{y} \mid X, \sigma^2, a) + \log p(a \mid \gamma^2)$$

We know the form of the log-likelihood as derived above.
$$\log p(a \mid \gamma^2) = \log
\left ( \frac{1}{\sqrt{2 \pi \gamma^2}} \exp \left \{ - \frac{1}{2 \gamma^2} a^2    \right \} \right )$$
$$= - \frac{1}{2} \log (2\pi \gamma^2) - \frac{a^2}{2 \gamma^2}$$ So the
log-posterior is:
$$\log p(a \mid D, \gamma^2) = - \frac{n}{2} \log (2 \pi \sigma^2)  - \frac{1}{2 \sigma^2}(\mathbf{y} - a \mathbf{x})^2 - \frac{1}{2} \log (2 \pi \gamma^2) - \frac{a^2}{2 \gamma^2}$$
$$\hat{a}_{\text{MAP}} = \mathop{\mathrm{arg\,min}}_a \frac{1}{2 \sigma^2} ((y - a \mathbf{x})^2 + \frac{\sigma^2}{\gamma^2} a^2)$$
$$= \mathop{\mathrm{arg\,min}}_a (y - a \mathbf{x})^2 + \lambda a^2$$

:::: centeredboxred
::: warning
**Warning: 1**. **MAP is not Bayesian**

We've essentially recovered ridge regression from assuming Gaussian
noise with a Gaussian prior. We can also recover Lasso regression from
assuming Laplace noise with a Laplace prior. However, MAP Estimation is
not a Bayesian method. The idea behind Bayesian inference is to
construct a probability distribution over the set of possible values for
$a$. MAP does not marginalize over the prior distribution so it does not
constitute a fully Bayesian treatment.
:::
::::

## Bayesian Inference

1.  **Prior**: We begin with some prior distribution $p(a)$. We have
    some dataset $D$ so we can compute the likelihood $p(D \mid a)$ to
    evaluate how well a particular setting of $a$ matches the dataset
    (the probability of observing $D$ given $a$).

2.  **Posterior**: We can then compute the posterior distribution.
    $$p(a \mid D) \propto p(D \mid A) \ p(a)$$

3.  **Bayesian Model Average**: Now, given our updated beliefs,
    $p(a \mid D)$, we want to compute the **predictive distribution**.
    Take our linear regression example - we have some dataset $D$ which
    you might think of as the training dataset. Now we have a new point
    $x_*$ and want to use our beliefs to construct a predictive
    distribution for $y_*$. In computing the predictive distribution, we
    again marginalize over the parameter $a$.
    $$p(y_* \mid x_*, \hat{\sigma}^2, \hat{\gamma}^2, \mathbf{y}, X) = \int_a p(y_*, a \mid x_*, \hat{\sigma}^2, \hat{\gamma}^2, \mathbf{y}, X) \ da$$
    $$= \int_a p(y_* \mid x_*, a, \hat{\sigma}^2) \ p(a \mid \hat{\gamma}^2, \mathbf{y}, X) \ da$$

    The Bayesian Model Average essentially weights all of the
    predictions of $y_*$ with some weight setting $a$ by the posterior
    probability of $a$. The posterior probability is the probability of
    $a$ after having observed the data (i.e. how well $a$ matches the
    data).

:::: centeredboxgreen
::: intuition
**Intuition Check: 3**. **Marginalizing over the Posterior**

Observe that when we were marginalizing over the data to obtain the
model evidence, we marginalized over the prior distribution (so each
likelihood was being weighted by our prior belief in it). When we
marginalize over $a$ now, we're marginalizing over the posterior
distribution. Now, each value of $y_*$ conditional on some value of $a$
is weighted by our updated belief in it, derived from the training data
$D$.

The idea with Bayesian Model Averaging is to *not* bet everything on a
single $a$ but to consider all possible parameter settings $a$ and also
to assume that the parameters are not completely determined by the data.
:::
::::

Generally, as the number of samples observed in $D$ increases, the
posterior distribution becomes increasingly sharp, centered around the
true parameter value $\hat{a}$. This can be seen in the following
diagram from Bishop (3.7). You can view the multi-variate Gaussian
describing the prior / posterior as concentrating and sharpening around
the true parameter value. Sequentially, we begin with a prior, derive a
posterior and the posterior becomes the new prior - this is the **online
nature** of sequential Bayesian learning.

**Bayesian Inference** is the process of determining the posterior and
thus, the Bayesian Model Average.

:::: centeredboxblue
::: example
**Example: 2**. **Frequentist vs Bayesian Urns**

Suppose we have $11$ urns $u_{[0, \dots, 10]}$. Each urn has $10$ balls.
The $i$th urn has $i$ black balls and $10 - i$ white balls. So the first
urn has $0$ black balls, the $10$th urn has $10$ black balls.

You are blindfolded and are given an urn. You draw $10$ balls with
replacement from the urn, one at a time. $3$ balls are black and $7$
balls are white. What is the probability of the next ball being black?
Specifically, is the probability greater than, equal to, or less than
$30 \%$?

The MLE solution is to say that you are in the $3$rd urn and that the
probability is equal to $30\%$. However, this doesn't incorporate
uncertainty about the fact that you may be in the $4$th or $5$th urn.
This uncertainty is included in the Bayesian Model Average. In reality,
the probability of the next ball being black is around $34\%$.
:::
::::

:::: centeredboxred
::: warning
**Warning: 2**. **MLE isn't necessarily optimal**

**Gaussian Mixture Models** are extremely expressive - they're dense in
the set of probability distributions and a mixture of $4$ or $5$
Gaussians is sufficient to model most classical statistical settings.

Specifically, we can gain infinite likelihood with a mixture of two
Gaussians. Note that this is density estimation so we're just predicting
$X$, rather than $y$ from $X$.
$$p(D \mid \mu_1, \mu_2, \sigma_1^2, \sigma_2^2) = \prod_{i = 1}^n \frac{w_1}{\sqrt{2 \pi \sigma_1^2}} \exp \left \{ - \frac{1}{2 \sigma_1^2} (x_i - \mu_1)^2 \right \} + \frac{w_2}{\sqrt{2 \pi \sigma_2^2}} \exp \left \{ - \frac{1}{2 \sigma_2^2} (x_i - \mu_2)^2   \right \}$$

By setting $\mu_1 = x_j$ and $\sigma_1^2 \to 0$, we can get essentially
infinite likelihood for $x_j$. The other Gaussian provides probability
mass to the other points to make this the product of infinity with other
non-zero numbers to get essentially infinite likelihood. This doesn't
mean that the model fits the data well - in fact, it fits it extremely
poorly. This is an example of **MLE overfitting**. To solve this, models
will often be re-initialized with the hope of falling into a local
minimum rather than a global minimum. This suggests that MLE is not a
good objective function at all for this specific case.
:::
::::

## Marginal Likelihood

There are $3$ different ways to approach any ML problem. In each case,
we have some function $f(x, w)$ that we want to estimate by inferring
information about $w$.

1.  Directly specific some loss function $L$ and minimize it with
    respect to $w$.

2.  Create a probabilistic story about how the data was generated
    (observation model) to form a likelihood $p(D \mid w)$ and maximize
    this with respect to $w$ (MLE or MAP).

3.  Take the Bayesian Approach. Compute the Bayesian Model Average,
    integrating over all possible parameter settings, weighted by their
    posterior probabilities for the predictive distribution.

Approaches $1$ and $2$ are very similar but the full Bayesian approach
is very different. This function is very flexible - $f(x, w)$ can be
done for regression / classification, unsupervised learning (as
**density estimation**) and the procedure for doing these is only
trivially different.

In **Bayesian Linear Regression**, consider the general form of a
**Linear Gaussian Model**: $$y = \mathbf{w}^T \phi(x) + \epsilon(x)$$
$$\epsilon(x) \sim \mathcal{N}(0, \sigma^2)$$
$$\mathbf{w} \mid \alpha \sim \mathcal{N}(0, \alpha^2 I)$$

Here, $\phi(x)$ is a vector of basis functions (for example,
$\phi(x) = [1, x, x^2, \dots, x^P]^T$). Clearly, $y(x)$ would be a
non-linear function but would be linear in $w$, the parameter we're
trying to estimate so we still consider this a linear model.

For a dataset $\mathcal{D} = \{(x_i, y_i)\}_{i = 1}^n$, we want to find
the marginal likelihood, $p(\mathbf{y} \mid X, \sigma^2, \alpha^2)$.
This is the denominator in Bayes equation, also known as the **partition
function**.

First, we marginalize over $\mathbf{w}$. $$\begin{equation}
    p(\mathbf{y} \mid X, \sigma^2, \alpha^2) = \int p(\mathbf{y} \mid \mathbf{w}, X, \sigma^2) p(\mathbf{w} \mid \alpha) \ d\mathbf{w}
\end{equation}$$

The prior and likelihood are both normal: $$\begin{equation}
    = \int \mathcal{N}(\mathbf{w} ; 0, \alpha^2 I) \mathcal{N}(\mathbf{y} ; \bm{\Phi} \mathbf{w}, \sigma^2 I) \ d\mathbf{w}
\end{equation}$$ $$\begin{equation}
    = c \int \exp \left \{- \frac{1}{2} \left ( \frac{\mathbf{w}^T \mathbf{w}}{\alpha^2} + \frac{(\mathbf{y} - \bm{\Phi} \mathbf{w})^T (\mathbf{y} - \bm{\Phi} \mathbf{w})}{\sigma^2}  \right )  \right \} \ d\mathbf{w}
\end{equation}$$

$$\begin{equation}
    = c \exp \left \{
    - \frac{1}{2}
    \left [
    \frac{\bm{w}^T \bm{w}}{\alpha^2} +
    \frac{1}{\sigma^2} (\bm{y}^T \bm{y} + \bm{w}^T \bm{\Phi}^T \bm{\Phi} \bm{w}  - 2 \bm{w}^T \bm{\Phi}^T \bm{y})
    \right ]
    \right \}
\end{equation}$$

$$\begin{equation}
\label{eq:lambda}
    = c \exp \left \{
    - \frac{1}{2}
    \left [
    \bm{w}^T (\frac{I}{\alpha^2} + \frac{\bm{\Phi}^T \bm{\Phi}}{\sigma^2}) \bm{w} + \frac{\bm{y}^T \bm{y}}{\sigma^2} - 2 \frac{\bm{w}^T \bm{\Phi}^T \bm{y}}{\sigma^2}
    \right ]
    \right \}
\end{equation}$$

Write the precision matrix,
$\Lambda = \frac{I}{\alpha^2} + \frac{\bm{\Phi}^T \bm{\Phi}}{\sigma^2}$.

$$\begin{equation}
\label{eq:5}
= c' \exp \left \{
- \frac{1}{2}
\left [
\bm{w}^T \Lambda \bm{w} - 2 \frac{\bm{w}^T \bm{\Phi}^T \bm{y}}{\sigma^2}
\right ]
\right \}
\end{equation}$$

Now we can really complete the square. First, we need to figure out the
mean vector. We have:

$$\begin{equation}
(\bm{w} - \bm{\mu})^T \Lambda (\bm{w} - \bm{\mu}) = \bm{w}^T \Lambda \bm{w} + \bm{\mu}^T \Lambda \bm{\mu} - 2 \bm{w}^T \Lambda \bm{\mu}
\end{equation}$$

By inspection, we have: $$\begin{equation}
\Lambda \bm{\mu} = \frac{\bm{\Phi}^T \bm{y}}{\sigma^2}
\end{equation}$$ $$\begin{equation}
    \bm{\mu} = \Lambda^{-1} \frac{\bm{\Phi}^T \bm{y}}{\sigma^2}
\end{equation}$$

Substituting in our mean vector, we have:

$$\begin{equation}
(\bm{w} - \bm{\mu})^T \Lambda (\bm{w} - \bm{\mu}) = \bm{w}^T \Lambda \bm{w} + \frac{\bm{y}^T \bm{\Phi}}{\sigma^2} \Lambda^{-1} \Lambda \Lambda^{-1} \frac{\bm{\Phi}^T \bm{y}}{\sigma^2} - 2 \bm{w}^T \Lambda \Lambda^{-1} \frac{\bm{\Phi}^T \bm{y}}{\sigma^2}
\end{equation}$$

$$\begin{equation}
= \bm{w}^T \Lambda \bm{w} + \frac{\bm{y}^T \bm{\Phi}}{\sigma^2} \Lambda^{-1} \frac{\bm{\Phi}^T \bm{y}}{\sigma^2} - 2 \bm{w}^T \frac{\bm{\Phi}^T \bm{y}}{\sigma^2}
\end{equation}$$ $$\begin{equation}
(\bm{w} - \bm{\mu})^T \Lambda (\bm{w} - \bm{\mu}) - \frac{\bm{y}^T \bm{\Phi}}{\sigma^2} \Lambda^{-1} \frac{\bm{\Phi}^T \bm{y}}{\sigma^2} = \bm{w}^T \Lambda \bm{w} - 2 \bm{w}^T \frac{\bm{\Phi}^T \bm{y}}{\sigma^2}
\end{equation}$$

Substituting in and putting the product of our MVGs back into the
integral, we have:

$$\begin{equation}
p(\bm{y} \mid X, \sigma^2, \alpha^2, \bm{z}) = c' \int \exp  \left \{ - \frac{1}{2} \left [
(\bm{w} - \bm{\mu})^T \Lambda (\bm{w} - \bm{\mu}) - \frac{\bm{y}^T \bm{\Phi}}{\sigma^2} \Lambda^{-1} \frac{\bm{\Phi}^T \bm{y}}{\sigma^2}
\right ]
\right \}
\end{equation}$$

$$\begin{equation}
= c'' \int \exp \left \{ - \frac{1}{2} (\bm{w} - \bm{\mu})^T \Lambda (\bm{w} - \bm{\mu}) \right \} d\bm{w}
\end{equation}$$

The exponential term is the unnormalized version of
$\mathcal{N}(\bm{w} ; \bm{\mu}, \Lambda^{-1})$. Since the integral over
$\bm{w}$ must be normalized, this must integrate to the normalizing
coefficient or:

$$\begin{equation}
\int \exp \left \{ - \frac{1}{2} (\bm{w} - \bm{\mu})^T \Lambda (\bm{w} - \bm{\mu}) \right \} d\bm{w} = \frac{(2 \pi)^{\frac{D}{2}}}{\det (\Lambda)^{\frac{1}{2}}}
\end{equation}$$

So we get: $$\begin{equation}
p(\bm{y} \mid X, \sigma^2, \alpha^2, \bm{z}) = c'' \frac{(2 \pi)^{\frac{D}{2}}}{\det (\Lambda)^{\frac{1}{2}}}
\end{equation}$$

$$\begin{equation}
p(\bm{y} \mid X, \sigma^2, \alpha^2, \bm{z}) = \frac{\exp \left \{- \frac{1}{2} \left [ \frac{\bm{y}^T \bm{y}}{\sigma^2} - \frac{\bm{y}^T \bm{\Phi}}{\sigma^2} \Lambda^{-1} \frac{\bm{\Phi}^T \bm{y}}{\sigma^2}   \right ] \right \}}{(2 \pi \sigma^2)^\frac{n}{2} \det (\Lambda)^{\frac{1}{2}} (\alpha^2)^{\frac{D}{2}}}
\end{equation}$$

$$\begin{equation}
\label{eq:e7}
= \frac{
\exp \left \{
- \frac{1}{2} \bm{y}^T \left [ \frac{I}{\sigma^2} - \frac{ \bm{\Phi}}{\sigma^2} \Lambda^{-1} \frac{\bm{\Phi}^T}{\sigma^2}   \right ] \bm{y}
\right \}
}{(2 \pi \sigma^2)^\frac{n}{2} \det (\Lambda)^{\frac{1}{2}} (\alpha^2)^{\frac{D}{2}}}
\end{equation}$$

Define
$\Lambda_0 = \frac{1}{\sigma^2} I - \frac{1}{\sigma^4} \bm{\Phi} \Lambda^{-1} \bm{\Phi}^T$.
Then, the numerator for [\[eq:e7\]](#eq:e7){reference-type="ref"
reference="eq:e7"} becomes the exponential for the unnormalized Gaussian
distribution $\mathcal{N}(0, \Lambda_0^{-1})$. First, by the Woodbury
Matrix identity, we'll find the covariance matrix
$\Sigma = \Lambda_0^{-1}$. We have: $$\begin{equation}
    (A + UCV)^{-1} = A^{-1} - A^{-1} U (C^{-1} + V A^{-1} U)^{-1} V A^{-1}
\end{equation}$$

Define $C = \frac{\Lambda^{-1}}{\sigma^4}$,
$U = - \bm{\Phi}, V = \bm{\Phi}^T, A = \frac{I}{\sigma^2}$. Then, we
have: $$\begin{equation}
    \Lambda_0^{-1} = \sigma^2 I - \sigma^2 I \bm{\Phi} (\sigma^4 \Lambda - \sigma^2 \bm{\Phi}^T \bm{\Phi})^{-1} \bm{\Phi}^T \sigma^2 I
\end{equation}$$

Substitute in $\Lambda$, we get: $$\begin{align}
    \Lambda_0^{-1} &= \sigma^2 I + \sigma^2 \bm{\Phi} (\sigma^4 (\frac{I}{\alpha^2} + \frac{\bm{\Phi}^T \bm{\Phi}}{\sigma^2}) - \sigma^2 \bm{\Phi}^T
    \bm{\Phi})^{-1} \bm{\Phi}^T \sigma^2 \\
    &= \sigma^2 I + \sigma^4 \bm{\Phi}( \frac{\sigma^4}{\alpha^2} I + \sigma^2 \bm{\phi}^T \bm{\Phi} - \sigma^2 \bm{\Phi}^T \bm{\Phi})^{-1} \bm{\Phi}^T \\
    &= \sigma^2 I + \sigma^4 \bm{\Phi} (\frac{\sigma^4}{\alpha^2} I)^{-1} \bm{\Phi}^T \\
    &= \sigma^2 I + \alpha^2 \bm{\Phi} \bm{\Phi}^T
\end{align}$$

This is the covariance matrix for the unnormalized probability
distribution in the numerator. We will now find it's determinant. By the
matrix determinant lemma, we have:

$$\begin{equation}
    \det(A + UWV^T) = \det(W^{-1} + V^T A^{-1} U) \det(W) \det(A)
\end{equation}$$ $$\begin{equation}
    \det(\sigma^2 I + \bm{\Phi} \alpha^2 I \bm{\Phi}^T) = \det(\frac{I}{\alpha^2} + \bm{\Phi}^T \frac{I}{\sigma^2} \bm{\Phi}) \det(\alpha^2 I) \det (\sigma^2 I)
\end{equation}$$

Notice that
$\Lambda = \frac{I}{\alpha^2} + \frac{\bm{\Phi}^T \bm{\Phi}}{\sigma^2}$.
So we can re-write: $$\begin{equation}
    \det(\Lambda_0^{-1}) = \det(\Lambda) (\alpha^2)^D (\sigma^2)^N
\end{equation}$$ $$\begin{equation}
\label{eq:lambda_0}
    \det(\Lambda_0^{-1})^{\frac{1}{2}} = \det(\Lambda)^{\frac{1}{2}}(\alpha^2)^{\frac{D}{2}} (\sigma^2)^{\frac{N}{2}}
\end{equation}$$

Recall our expression for the marginal likelihood (re-written here).

$$\begin{equation}
     p(\bm{y} \mid X, \sigma^2, \alpha^2, \bm{z}) = \frac{
\exp \left \{
- \frac{1}{2} \bm{y}^T \left [ \frac{I}{\sigma^2} - \frac{ \bm{\Phi}}{\sigma^2} \Lambda^{-1} \frac{\bm{\Phi}^T}{\sigma^2}   \right ] \bm{y}
\right \}
}{(2 \pi)^{\frac{N}{2}} (\sigma^2)^\frac{n}{2} \det (\Lambda)^{\frac{1}{2}} (\alpha^2)^{\frac{D}{2}}}
\end{equation}$$

Substituting in our result from
[\[eq:lambda_0\]](#eq:lambda_0){reference-type="ref"
reference="eq:lambda_0"}, we get:

$$\begin{equation}
    p(\bm{y} \mid X, \sigma^2, \alpha^2, \bm{z}) = \frac{
\exp \left \{
- \frac{1}{2} \bm{y}^T \Lambda_0 \bm{y}
\right \}
}{(2 \pi)^{\frac{N}{2}} \det(\Lambda_0^{-1})^{\frac{1}{2}}}
\end{equation}$$

In other words, this is a fully normalized Gaussian distribution, i.e.
$\mathbf{y} \mid X, \sigma^2, \alpha^2, \bm{z} \sim \mathcal{N}(0, \Lambda_0^{-1})$.

Define $\bm{\Sigma}_0 = \Lambda_0^{-1}$ to be the covariance matrix of
the marginal likelihood.

Thus, the **log marginal likelihood** is:

$$\begin{equation}
\log p(\mathbf{y} \mid \mathbf{z}, X, \alpha^2, \sigma^2) = - \frac{1}{2} \mathbf{y}^T \bm{\Sigma}_0 \mathbf{y} - \frac{1}{2} \log (\det(\bm{\Sigma}_0^{-1})) - \frac{N}{2} \log (2 \pi)
\end{equation}$$

This compartmentalizes nicely into a data fit and model complexity term.

The data fit term is $\mathbf{y}^T \bm{\Sigma}_0 \mathbf{y}$ and the
model complexity term is $\log (\det(\bm{\Sigma}_0^{-1})$). The data fit
term measures the discrepancy between the data your model expects (in
the covariance matrix) and the data itself (in $\mathbf{y}$). The
determinant of the covariance matrix measures the volume of probability
that can be explained by the model - so it penalizes more complex
models. This is a built-in Occam's razor.

We can perform *Marginal Likelihood Optimization* or *Type 2 MLE* to
learn hyperparameters - this is much more efficient than grid search
where the number of runs grows exponentially with the number of
hyperparameters. It's very useful but doesn't answer the exact question
we want - it answers the likelihood of the data under all parameter
settings when we want the probability of the model class itself.

## Predictive Distribution

Let's say we have optimized the marginal likelihood to obtain point
estimates $\hat{\alpha}^2, \hat{\sigma}^2$. Our ultimate goal is to make
predictions. With frequentist statistics, we make point estimate
predictions but with a Bayesian toolbox, we can construct an entire
distribution of predictions. We want to find
$p(y_* \mid \mathcal{D}, x_*, \hat{\sigma}^2, \hat{\alpha}^2)$. We can
marginalize over the weights.

$$\begin{equation}
    p(y_* \mid \mathcal{D}, x_*, \hat{\sigma}^2, \hat{\alpha}^2) = \int p(y_* \mid \mathbf{w}, x_*, \sigma^2) \ p(\mathbf{w} \mid \mathcal{D}, \hat{\alpha}^2) \ d\mathbf{w}
\end{equation}$$

First, we need to find the posterior distribution. We have:

$$\begin{equation}
    p(\mathbf{w} \mid \mathcal{D}) \propto p(\mathcal{D} \mid \mathbf{w}) \ p(\mathbf{w} \mid \alpha)
\end{equation}$$

where $\mathbf{w} \mid \alpha \sim \mathcal{N}(0, \alpha^2 I)$. However,
we already found the posterior distribution as the integrand in the
marginal likelihood. We found that
$\mathbf{w} \mid \mathcal{D} \sim N(\bm{\mu}, \Lambda^{-1})$ where
$\bm{\mu} = \Lambda^{-1} \frac{\bm{\Phi}^T \mathbf{y}}{\sigma^2}$ and
$\Lambda = \frac{I}{\alpha^2} + \frac{\bm{\Phi}^T \bm{\Phi}}{\sigma^2}$.
In finding the predictive distribution, we can avoid computing the
integral through the laws of total expectation and variance.

$$\begin{equation}
    E[y_* \mid x_*, \hat{\sigma}^2, \hat{\alpha}^2, \mathcal{D}] = E_{\mathbf{w} \mid \hat{\alpha}^2, \mathcal{D}}[E[y_* \mid \mathbf{w}, x_*, \hat{\sigma}]]
\end{equation}$$ $$\begin{equation}
    E[y_* \mid \mathbf{w}, x_*, \hat{\sigma}] = E[\mathbf{w}^T \phi(x) + \mathcal{N}(0, \hat{\sigma}^2)] = \phi(x)^T E[\mathbf{w}]
\end{equation}$$

From the posterior mean, we know that: $$\begin{equation}
    \phi(x)^T E[\mathbf{w}] = \bm{\mu}^T \phi(x)
\end{equation}$$

The expectation of a scalar is a scalar so this is the predictive mean.
For the predictive variance, we have: $$\begin{equation}
    \mathop{\mathrm{Var}}(y_* \mid x_*, \hat{\alpha}^2, \hat{\sigma}^2, \mathcal{D}) = E_{\mathbf{w} \mid \hat{\alpha}^2, \mathcal{D}}[\mathop{\mathrm{Var}}(y_* \mid x_*, \hat{\sigma}^2, \mathbf{w})] + \mathop{\mathrm{Var}}(E_{\mathbf{w} \mid \hat{\alpha}^2, \mathcal{D}}[y_* \mid x_*, \mathbf{w}, \hat{\sigma}^2])
\end{equation}$$

We have
$y_* \mid x_*, \hat{\sigma}^2, \mathbf{w} \sim \mathcal{N}(0, \hat{\sigma}^2)$
so the variance is $\hat{\sigma}^2$ so the expectation of the variance
is just $\hat{\sigma}^2$. The variance of the expectation is just
$\mathop{\mathrm{Var}}[E_{\mathbf{w} \mid \hat{\alpha}^2, \mathcal{D}}[\mathbf{w}^T \phi(x_*) + \epsilon(x)]]$.
The expectation just becomes $\mathbf{w}^T \phi(x_*)$. The variance of
this term is thus
$\phi(x_*)^T \mathop{\mathrm{Var}}[\mathbf{w}] \phi(x_*)$. However, the
variance of this term is just the covariance matrix so we have:

$$\begin{equation}
    y_* \mid x_*, \mathcal{D}, \hat{\sigma}^2, \hat{\alpha}^2 \sim \mathcal{N}(y(x_*) ; m_*, \nu_*)
\end{equation}$$ $$\begin{equation}
    m_* = \bm{\mu}^T \phi(x)
\end{equation}$$ $$\begin{equation}
    \nu_* = \hat{\sigma}^2 + \phi(x_*)^T \Lambda^{-1} \phi(x_*)
\end{equation}$$

:::: centeredboxblue
::: example
**Example: 3**. **Sanity Check**

$m_* = \bm{\mu} ^T \phi(x)$: this is a reasonable answer for our mean as
it's the inner product of our posterior mean (expected values of
parameter after having observed $\mathcal{D}$) with our parameters. The
variance $\nu_* = \sigma^2 + \phi(x_*)^T \Lambda^{-1} \phi(x_*)$ so we
have noise variance $\sigma^2$ and the uncertainty over the parameters.

Consider the extreme. Suppose we have a ton of noise, i.e.
$\sigma \to \infty$. Then, the posterior mean goes to $0$ and variance
goes to $\alpha^2 I$. This is the prior. In other words, if there's a
ton of noise and we can't infer information from the likelihood, our
posterior becomes our prior. If $\alpha \to 0$, we have zero variance
over weights and our distribution becomes peaked with mean $0$ and
variance $0$. Our prior is too strong.
:::
::::

To summarize:

1.  We find the predictive distribution by marginalizing over different
    parameter settings and weighting them by the posterior probability.
    $$\begin{equation}
        p(y(x_*) \mid \alpha^2, \sigma^2) = \int p(y(x_*) \mid \mathbf{w}, \sigma^2) p(\mathbf{w} \mid \alpha^2) \ d\mathbf{w}
    \end{equation}$$

2.  We optimize the hyperparameters by maximizing the marginal
    likelihood of the data. $$\begin{equation}
        \hat{\alpha}^2, \hat{\sigma}^2 = \mathop{\mathrm{arg\,max}}_{\alpha^2, \sigma^2} p(D \mid \alpha^2, \sigma^2) = \mathop{\mathrm{arg\,max}}_{\alpha^2, \sigma^2} \int p(D \mid w, \sigma^2) p(w \mid \alpha^2) \ d\mathbf{w}
    \end{equation}$$

3.  We make predictions
    $p(y(x_*) \mid \hat{\alpha}^2, \hat{\sigma}^2), x_*, \mathcal{D}$.

However, this does not constitute a fully Bayesian treatment. Fully
Bayesian methods do not optimize hyperparameters but rather marginalize
over them (Bayesian Model Average). We don't bet everything on a single
setting of parameters because we have uncertainty over our parameters
even after we've observed data. In this procedure, we're optimizing
$\hat{\sigma}^2, \hat{\alpha}^2$ and conditioning our predictive
distribution on those values. But our posterior over $\alpha^2$
shouldn't be a point mass but rather a distribution.

What we really want to find is $p(y(x_*) \mid \mathbf{y}, X, x_*)$.

$$p(y(x_*) \mid \mathbf{y}, X, x_*) = \int p(y(x_*) \mid \alpha^2, \sigma^2) p(\alpha^2, \sigma^2 \mid D) \ d\alpha^2  \ d\sigma^2$$

We apply our reasoning for marginalizing over $\mathbf{w}$ to also
marginalize over $\alpha^2, \sigma^2$. The posterior distribution over
$\alpha^2, \sigma^2$ is thus:

$$p(\alpha^2, \sigma^2 \mid D) \propto p(D \mid \alpha^2, \sigma^2) p(\alpha^2, \sigma^2)$$

The earlier $3$-step procedure is a reasonably Bayesian approach as it's
doing the BMA over low-level parameters $\mathbf{w}$ which are highly
influential on our predictions. Optimizing the marginal likelihood also
has a automatic model complexity penalty which has less risk of
overfitting so it's not a completely non-Bayesian approach. This is
called **empirical Bayes**.

The prior distribution has hyperparameters and we theoretically want to marginalize over those. However, this just constitutes moving one layer up a hierarchy of priors, hyperparameters for the priors, hyperparameters for those hyperparameters, etc. When do we stop doing integration? Isn't there infinite recursion? The intuitive answer is that at some point, it doesn't matter too much. If $w$ are the parameters, $\alpha^2, \sigma^2$ are the hyper-parameters, so on and so forth, the setting of those don't influence our predictions as much as we have so much uncertainty.
